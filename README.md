# Student Average Calculator

This is a simple Python project that reads student scores from a file and calculates the average. It also checks whether the student has passed or failed based on the average score.

## What the program does
- Reads scores from a text file (`Scores.txt`)
- Cleans and converts the data into integers
- Calculates the average score
- Determines if the student has passed or failed

## Concepts used
- File handling in Python
- Lists
- map() function
- Lambda functions
- Conditional statements (if/else)

## How to run the project
- Make sure Python is installed on your computer
- Place `main.py` and `Scores.txt` in the same folder
- Run the program using:

```bash
python main.py
